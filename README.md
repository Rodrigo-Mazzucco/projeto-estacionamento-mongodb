# Projeto Estacionamento MongoDB

API REST desenvolvida com Node.js, Express e MongoDB para gerenciamento de proprietários e seus veículos.

## Funcionalidades
- Cadastrar proprietário
- Listar proprietários
- Buscar proprietário por ID
- Atualizar proprietário
- Excluir proprietário
- Gerenciar veículos vinculados ao proprietário

## Tecnologias
- Node.js
- Express
- MongoDB
- Mongoose
- Cors
- Body-Parser
